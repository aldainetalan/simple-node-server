Early World Wide Web




The World Wide Web started through the idea of J.C.R. Licklider that network of computers can be connected together by “wide-band communication lines”.

After the creation of the ARPANET in 1969, many universities and hosts were added to the APRANET. The system turned to be more stabilized where by 1981, there were over 200 hosts on the system.

The World-Wide Web began in March 1989 at CERN, now called “European Laboratory for Particle Physics.” The invention of the web is credited to Tim Berners-Lee, a contactor at CERN. In 1990, Berners-Lee developed the skeletal outline of the internet, including a Web browser and Web Server. The web failed to gain great popularity until the release of Mosaic Web Browser in 1993. On April 30, 1993, CERN released the web to the public with no charges. Further development on the web was done until Berners-Lee founded the World Wide Consortium (W3C) in 1994 that allowed users to access and use the web with much more ease.
